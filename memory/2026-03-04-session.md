## Architecture Notes

### Calculator Pattern
All new calculators follow:
```
1. useState for inputs
2. Calculate results from inputs
3. Render input form + result cards
4. SaveToPortfolioButton with projectData
5. Analysis/interpretation section
```

### Currency Support
All calculators support: IDR, USD, AUD, EUR

### Portfolio Integration
- Projects save via usePortfolio hook
- Fields: projectName, location, totalInvestment, roi, avgCashFlow, breakEvenMonths, investmentScore, currency
- Stored in localStorage (temporary, needs backend for enterprise)

## Known Limitations
1. **No backend** — Using localStorage only (OK for MVP, needs DB for enterprise)
2. **No user auth** — All projects in browser
3. **No email reports** — SendEmail component exists but not hooked up
4. **No scenarios** — Can save projects but not variations
5. **No sensitivity analysis** — No what-if/Monte Carlo yet
